File list:
CYC_HWFET.mat: EPA highway cycle
CYC_UDDS.mat: EPA city cycle
CYC_US06.mat: US 06 cycle

prius_sim_data.m: data for vehicle, driver, battery and all three power components
prius_sim_init.m: initialize the rule based control parameters and run the simulation with different driving cycles (optional). To run the simulation, just run this file.
prius_sim_plot.m: make the plots after the simulation

prius_rulebased.mdl: the full model with controller and vehicle.
prius_nocontrol: the model without controller.

priusdynamics.ppt: the slides explain the equations.
